Quick Start

Clone the repo, install dependencies, copy the example env file, and open the dashboard.

Terminal · maestRoRun setup 
# clone and install
git clone https://github.com/yunusemreyildiz/maestrorun.git
cd maestrorun
npm install
cp .env.example .env
npm run dashboard

# then open
http://127.0.0.1:5173

Then open http://127.0.0.1:5173. Real device execution also needs Maestro CLI, Java 17+, and platform tooling from the dashboard setup guide.
